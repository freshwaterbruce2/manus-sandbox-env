# Manus Sandbox Environment

This repository is the **official workspace** for the Manus AI agent operating in the `freshwaterbruce2` GitHub environment.

## Purpose

This repo serves as:

- A persistent store for scripts, tools, and utilities developed by Manus
- A version-controlled log of automation tasks and experiments
- A collaboration bridge between Manus and the repository owner

## Structure

```bash
manus-sandbox-env/
├── README.md          # This file
├── scripts/           # Utility and automation scripts
├── tools/             # Reusable tools and helpers
├── experiments/       # One-off experiments and tests
└── docs/              # Documentation and notes
```
